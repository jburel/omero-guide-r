R Gateway
=========

This section shows how to install and use the R Gateway to analyze data.
Jupyter notebooks are available to demonstrate some of the functionalities available.

Follow the instructions in the
:download:`README.md <https://github.com/ome/omero-guide-r/blob/master/README.md>` file to run the notebooks either locally or on `mybinder.org <https://mybinder.org/v2/gh/ome/omero-guide-r/master?filepath=notebooks>`__.
