R Gateway
=========

This section shows how to install and use the R Gateway to analyze data.
Jupyter notebooks are available to demonstrate some of the functionalities available.


**Run the notebooks**
---------------------

Follow the instructions in 
:download:`README <../README.md>` to run the notebooks either locally or `on mybinder.org <https://mybinder.org/v2/gh/ome/omero-guide-r/master?filepath=notebooks>`_.

**Contribute**
--------------

Changes to the documentation or the materials should be made directly in the `omero-guide-r repository <https://github.com/ome/omero-guide-r>`_.
